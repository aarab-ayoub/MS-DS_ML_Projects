# Course TPs — Deep Learning

Practical work notebooks from the Deep Learning module. Each TP is self-contained and can be run independently.

| File | Topic |

| [TP01_CC_exam.ipynb](TP01_CC_exam.ipynb) | Exam / graded practical |
| [TP02_supervised_learning.ipynb](TP02_supervised_learning.ipynb) | Supervised learning models |
| [TP03_more_ml_models.ipynb](TP03_more_ml_models.ipynb) | Additional ML models & techniques |
| [TP04_perceptron_spam_detection.ipynb](TP04_perceptron_spam_detection.ipynb) | Perceptron — spam email detection |
| [TP05_ann_gradient_descent.ipynb](TP05_ann_gradient_descent.ipynb) | 2-layer ANN, gradient descent, dataset generation |
| [TP11_cnn_image_recognition_cifar10.ipynb](TP11_cnn_image_recognition_cifar10.ipynb) | CNN image recognition on CIFAR-10 |
